Definition runestone.h:121

Macro for defining the available operand types.

This macro defines the various operand types used in Runestone IR instructions. It is used to generate the rs_operand_type_t enum and to define constructor macros for different operand types. Basic block reference.

enum rs_operand_type_t

Enumerates the possible operand types for an instruction.

This enum defines the types of operands that can be used in Runestone IR instructions, such as integer constants, virtual registers, addresses, and basic block references.
