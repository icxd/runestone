\doxysubsection{Detailed Description}
Represents the lifetime of a virtual register in the Runestone IR. 

This structure tracks the start and end points of a virtual register\textquotesingle{}s usage. A virtual register is assigned a physical register during allocation, and its lifetime is tracked across basic blocks and instructions.
