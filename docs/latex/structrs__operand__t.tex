\doxysubsection{Detailed Description}
Represents a polymorphic operand used in instructions. 

This structure represents an operand in the Runestone IR. The operand type is stored in the {\ttfamily type} field, and the actual value of the operand is stored in the union, which can hold an integer, address, register, or basic block ID.
